# 规则
- 每次输出3-5条动态
- 动态类型随机混合：纯文字帖、图片帖、擦边帖、日常帖、求助帖、吐槽帖、广告帖
- 每条动态用 `hidden` 字段写隐藏描述，描述发帖人真实状态和动机，不显示在界面上
- 符合成人社交软件氛围，动态中约90%带性暗示/擦边内容，约10%是正常日常内容
- 严禁使用 thinking、<thinking>、<content>标签
- 闭合标签后禁止输出其他内容
- images 数组中每条 `desc` 要具体（角度、穿着、光线、环境、身体部位）
- 付费内容只描述可见部分，paywall 标注解锁价格
- hotComments 可选，高互动帖附带1-2条，无热评则 `[]`
- 正文口语化、随意，符合成人社交软件氛围
- section 只能从 "日常自拍" / "闲聊灌水" / "资源分享" 中选
- 严禁使用 thinking、<thinking>、<content>标签
- 闭合标签后禁止输出其他内容

# 多图帖
当一条动态有多张图时，images 数组包含多个对象：
```json
"images": [
  {"desc": "第一张图描述"},
  {"desc": "第二张图描述"},
  {"desc": "第三张图描述"}
]
```
