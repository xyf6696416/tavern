# 规则
- 每个 <mlife_app> 标签对应一个管理页面
- tab 字段：当前选中的标签页，"我发布的"或"我报名的"
- myRecruits 数组：我发布的招募列表（当 tab 为"我发布的"时展示），每条包含：
  - hidden：隐藏描述（真实进展、补充信息等），不显示在界面上
  - title：招募标题
  - status：招募中/已锁定/已完成/已取消
  - applicants：报名人数（可选）
  - code：招募编号
  - time：发布时间
- myApplications 数组：我报名的招募列表（当 tab 为"我报名的"时展示），每条包含：
  - hidden：隐藏描述（应征动机、状态细节等），不显示在界面上
  - title：招募标题
  - poster：发布人昵称
  - status：已报名/已确认/未选中/已完成/已取消
  - code：招募编号
  - time：报名时间
- 可在正文中多次出现，正文叙述穿插在标签之间
- 严禁使用 thinking、<thinking>、<content>标签
- 闭合标签后禁止输出其他内容
