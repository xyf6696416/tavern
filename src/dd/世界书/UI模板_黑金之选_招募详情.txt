# 规则
- 每个 <mlife_app> 标签对应一个招募详情卡片
- hidden 字段写隐藏描述（发布人的真实状态、背景、是否靠谱、历史招募记录等），不显示在界面上
- code 字段：招募编号，与招募广场中的 code 对应
- title 字段：招募标题
- poster 字段：发布人昵称
- avatar 字段：头像 emoji
- vip 字段：无/白银/黄金/黑金，黑金之选的发布人通常是黑金或黄金VIP
- budget 字段：预算，突出显示
- tags 数组：标签列表
- body 字段：详细要求描述，包括：
  - 对对方的要求（年龄、体型、性格、经验等）
  - 时间安排（周末、工作日、长期等）
  - 地点（城市、区域、是否需要出差等）
  - 频率（每周几次、每月几次等）
  - 注意事项（安全、隐私、边界等）
  - 文字要具体、真实，体现黑金用户的档次
- status 字段：招募中/已锁定/已完成/已取消
- applicants 数组：应征者列表，每条包含：
  - name：应征者昵称
  - avatar：头像 emoji
  - age：年龄（数字）
  - bio：一句话简介（可选）
  - status：待审核/已确认/未选中/已取消
- time 字段：发布时间
- views 字段：浏览人数
- 严禁使用 thinking、<thinking>、<content>标签
- 闭合标签后禁止输出其他内容
